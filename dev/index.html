1 C++

  • Indexing, code completion: ctags and YouCompleteMe
  • Testing: cppunit and googletest
  • Documenting: doxygen
  • Code formatting: clang-format
  • Static analysis: clang-tidy and IWYU
  • Dynamic analysis: valgrind and sanitizers
  • Build system: automake and cmake
  • Debugging: gdb, Visual Studio

More details:

1.1 C++11

  • LLVM docs

  • Features that I saw working with MSVC 2012, GCC 4.8 (ideally >= 4.6) and Clang 3.4 (ideally >=3.1):

    • Range-based for-loop
    • Lambdas (but not lambdas with default arguments)
    • auto type deduction
    • Strongly-typed and forward declarable enums

1.2 C++17

Features that I saw working in LibreOffice:

  • std::string_view

  • structured bindings

  • inline variables

  • fallthrough

  • nodiscard

  • class template argument deduction

  • deeplinks: wg21.link

  • html deeplinks: eel.is

2 Go

4 Rust

  • Indexing, code completion: rusty-tags and YouCompleteMe
  • Testing: cargo-test
  • Documenting: rustdoc
  • Code formatting: rustfmt
  • Static analysis: rust-clippy
  • Dynamic analysis: valgrind and sanitizers (though usually neither of these are needed)
  • Build system: cargo-build
  • Debugging: rust-gdb
